Analise-de-custo-Hospitalar

Sobre o Projeto

Este projeto realiza uma análise completa de custos hospitalares com o objetivo de transformar dados brutos em informações estratégicas.

Através de técnicas de análise de dados e visualização, o projeto permite identificar padrões de gastos, setores mais custosos e oportunidades de otimização, contribuindo para uma gestão hospitalar mais eficiente e orientada por dados.


Resultados do Projeto : 

- Análise de custos por setor  
- Comparação entre custos diretos e indiretos  
- Dashboards interativos e visuais  
- Identificação de gargalos financeiros  
- Projeto aplicado ao contexto real da saúde  



Tecnologias Utilizadas : 

Python (Pandas, NumPy)  
Jupyter Notebook  
Power BI / Excel  
Git & GitHub  
CSV / Excel

Conclusão : 

Após a realização das análises dos dados hospitalares, foi possível identificar que o procedimento que mais gerou custos no setor de emergência foi a cirurgia. Esse fator impactou diretamente no resultado geral, tornando a emergência o setor mais oneroso dentro do hospital.

Além disso, as métricas desenvolvidas ao longo do projeto permitiram uma visualização mais clara do uso do caixa, evidenciando padrões de gastos e auxiliando na compreensão da distribuição dos custos. Dessa forma, os resultados obtidos não apenas contribuem para o monitoramento financeiro, mas também servem como base para a possível tomada de decisões futuras, visando otimização de recursos e melhoria na gestão hospitalar.
