Analise-de-custo-Hospitalar

Sobre o Projeto

Este projeto realiza uma análise completa de custos hospitalares com o objetivo de transformar dados brutos em informações estratégicas.

Através de técnicas de análise de dados e visualização, o projeto permite identificar padrões de gastos, setores mais custosos e oportunidades de otimização, contribuindo para uma gestão hospitalar mais eficiente e orientada por dados.


Resultados do Projeto : 

- Análise de custos por setor  
- Comparação entre custos diretos e indiretos  
- Dashboards interativos e visuais  
- Identificação de gargalos financeiros  
- Projeto aplicado ao contexto real da saúde  



Tecnologias Utilizadas : 

Python (Pandas, NumPy)  
Jupyter Notebook  
Power BI / Excel  
Git & GitHub  
CSV / Excel
